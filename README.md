# Боевые Роботы (Mech Commander Prototype)

Тактическая RTS про отряд боевых мехов, зачищающих один процедурно сгенерированный уровень за другим.

🎮 **Играть:** [Yandex Игры](https://yandex.ru/games/app/555248?draft=true&lang=ru)

## О проекте

Каждая миссия — либо уничтожить всех врагов на карте, либо дойти до точки эвакуации. После победы игрок выбирает одно улучшение для меха (HP, броня, урон или скорострельность) и переходит на следующий, более сложный уровень. Каждый третий уровень заканчивается схваткой с усиленным боссом — а после победы над ним в отряд может влиться ещё один мех. Забег длится до первого поражения, а затем начинается заново — с новым отрядом и новыми картами.

## Основные механики

- **Процедурная генерация уровней** — форма играбельной зоны, размер карты, количество и расположение противников считаются по фактической площади проходимой земли (NavMesh), а не по абстрактному квадрату.
- **RTS-управление отрядом** — выбор юнитов рамкой, группы (1–9 / Ctrl+1–9), приказы Move / Attack-Move / Hold / Patrol / Guard, стойки Passive / Defensive / Aggressive.
- **Три типа оружия и два режима стрельбы** — энергетическое / баллистическое / ракетное, залповый огонь в упор (Alpha Strike) или прицельная стрельба на дистанции (Chain Fire), с системой перегрева.
- **Прогрессия забега** — апгрейды между уровнями, растущая сложность противников (+10% к характеристикам за уровень), боссы каждый 3-й уровень, пополнение отряда новыми мехами.
- **Кастомизация меха** — выбор модели, цвета, оружия и стойки перед стартом забега.
- **Монетизация под Yandex Games** — интеграция Yandex Games SDK (interstitial между уровнями, rewarded-бонусы).

## Технологии

- Unity 6000.5.1f1, Universal Render Pipeline
- Новый Input System
- AI Navigation (NavMesh) для процедурной генерации и перемещения юнитов
- Timeline (вступительный/завершающий ролик)
- WebGL-сборка под Yandex Games (кастомный темплейт, интеграция Yandex Games SDK через .jslib)

## Разработка

Начиная с определённого этапа реализация в Unity велась через [Claude Code](https://claude.com/product/claude-code) и [MCP for Unity](https://github.com/CoplayDev/unity-mcp): задачи формулировались как промпты, Claude Code вносил изменения в сцены/скрипты напрямую через MCP-соединение с редактором.

## Управление

| Действие | Клавиша |
|---|---|
| Выбор юнита / рамка | ЛКМ |
| Движение / атака по цели | ПКМ |
| Attack-move | A |
| Удерживать позицию | H |
| Патрулирование (+клик) | P |
| Сопровождение союзника (+клик) | G |
| Смена стойки | X |
| Выбрать / создать группу | 1–9 / Ctrl+1–9 |
